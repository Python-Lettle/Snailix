# Snailix

## Abstract

Snailix is a simple and easy-to-use shell. More interesting features will be supported in the future.

## Usage

```shell
$ make run
```

## Features



### Basic

- [x] Boot & Loader
- [x] Screen (80x25 text mode)
- [ ] Interrupt
- [ ] System call
- [ ] Clock
- [ ] Time
- [ ] Memory Manage
- [ ] Process

-   [ ] Thread
-   [ ] Keyboard

### Major functions

-   [ ] Command line



### Lib

-   [ ] Linked List
-   [ ] Bitmap
-   [ ] String
