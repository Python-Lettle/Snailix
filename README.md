# Snailix

## Abstract

Snailix is a simple and easy-to-use shell. More interesting features will be supported in the future.

## Usage

```shell
$ make run
```

## Features



### Basic

- [x] Boot & Loader
- [x] Screen (80x25 text mode)
- [x] Interrupt
- [x] System call
- [x] Clock
- [x] Time
- [ ] Memory Manage
- [ ] Process

-   [x] Thread
-   [x] Keyboard

### Major functions

-   [ ] Command line



### Lib

-   [x] Linked List
-   [x] Bitmap
-   [x] String
